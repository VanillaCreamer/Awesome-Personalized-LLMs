# Weekly Email Digest Template

Subject:
`Weekly Personalized-LLMs arXiv Digest - YYYY-MM-DD`

Body structure:

```text
Weekly Personalized-LLMs arXiv Digest
Date: YYYY-MM-DD
Source: arXiv only

Accepted papers: N

1. Title
Category: ...
Link: https://arxiv.org/abs/....
Summary:
<Abstract or short intro>

2. Title
Category: ...
Link: https://arxiv.org/abs/....
Summary:
<Abstract or short intro>
```

Notes:
- Email includes titles and abstracts/brief descriptions.
- PR does not include abstracts.
- If no new papers are accepted in a week, send a short "no qualified new papers this week" digest.
